# Packaging Scaffolds

This folder contains release packaging templates that map to the GitHub release artifacts produced by CI.

## Inputs

- Release tag/version, for example `v1.0.0` and `1.0.0`
- `SHA256SUMS.txt` generated by the `release` job in `.github/workflows/ci.yml`
- Optional helper: `scripts/render_release_packaging.sh <version> <sha256sums_file> [output_dir] [release_date]`

## Homebrew (tap)

- Template: `packaging/homebrew/fixdecoder.rb.tmpl`
- Replace `__VERSION__` and all `__SHA256_*__` placeholders.
- Copy the rendered file to your tap repo as `Formula/fixdecoder.rb`.

## WinGet

- Templates:
  - `packaging/winget/StephenLClarke.Fixdecoder.yaml.tmpl`
  - `packaging/winget/StephenLClarke.Fixdecoder.locale.en-US.yaml.tmpl`
  - `packaging/winget/StephenLClarke.Fixdecoder.installer.yaml.tmpl`
- Replace `__VERSION__`, `__RELEASE_DATE__`, and checksum placeholders.
- Submit rendered manifests to `microsoft/winget-pkgs`.
